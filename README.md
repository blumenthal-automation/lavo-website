## How to Use

### Deploy
1. Copy `index.html` to your web server
2. Serve via HTTPS recommended
3. No build process or dependencies needed

### Customize
Edit the `index.html` file directly:
- Content: Update text in section divs
- Colors: Modify CSS variables in `:root` style block
- Services/Case Studies: Add or remove cards as needed
- Contact information: Update footer links

### Performance Testing
- Open in Chrome DevTools
- Test responsive design at 1440px, 768px, 480px
- Check animation performance on scroll
- Verify all interactive elements work
